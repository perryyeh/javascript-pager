# js 分页样式

本js分页样式目前支持html，asp，php，jsp等，可以是类似2005042155_1.html，2005042155_2.html...2005042155_10.html这样的静态分页，也支持自定义参数，比如xxx.asp?yy=5等。使用模板技术，即使里面有＃等特殊字符也不怕了：）其余参数为自动获取继承。
